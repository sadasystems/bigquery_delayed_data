# **bigquery-workspace**
### Just some helper scripts for BigQuery...
Find in this directory a few helper scripts for delayed data in BigQuery
### General Format:  
- **Header**: script name  
- **Text underneath**: Information on how to use the script


## **check_empty_table.py**
Within data warehouses, there is often a need to determine how to handle **delayed data**. check_empty_table.py can be used as a helper script within your data pipelines to check if a table within BigQuery is empty (0 rows).

## **starter.py**
Just some template code for getting started with the BigQuery Python Client Library.
